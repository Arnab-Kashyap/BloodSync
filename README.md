# 🩸 BloodSync

A web application to find blood donors during emergencies — fast, smart, and reliable.

## What it does
- Search donors by blood group and location (India-wide)
- Smart donor ranking based on availability, distance & donation history
- Emergency request system to alert nearby donors
- AI-powered chatbot for blood donation guidance

## Tech Stack
- **Frontend** — React + Vite + Tailwind CSS
- **Backend** — Node.js + Express
- **Database** — MongoDB Atlas
- **AI** — Anthropic Claude API
